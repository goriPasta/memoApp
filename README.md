# Memo System (v0)

軽量な個人メモシステムのひな形です。

## 特徴
- **マルチデバイス対応**: PC、iPhoneのブラウザから利用可能。
- **データ永続化**: ブラウザの `localStorage` を使用。
- **検索機能**: キーワードおよびタグによるフィルタリング。
- **シンプル設計**: GitHub API への差し替えが容易なストレージ層。

## ファイル構成
- `index.html`: UI構造
- `style.css`: スタイル（レスポンシブ）
- `app.js`: アプリケーションロジック
- `storage.js`: データストレージ層（localStorage）
- `data/notes.json`: 初期のダミーデータ

## 使い方
1. このリポジトリをブラウザで開く（GitHub Pagesなど）。
2. メモを入力し「保存」をクリック。
3. 検索ボックスでメモを絞り込み。

## 将来の拡張
- GitHub API 連携によるデータの永続化と同期。
- AIによる自動タグ付け・要約。
- CLIツールによるデータ操作。
